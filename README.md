# KAPAKA AI — Secure Console

A futuristic **Cyberpunk-style AI control console login interface** built using **pure HTML, CSS, and JavaScript**.

This project simulates a **secure AI operations gateway** with animated UI elements, terminal telemetry, biometric verification animation, and a hacker-style HUD interface.

---

## Overview

**KAPAKA AI Secure Console** is a front-end concept for an AI control panel login system designed with a **cyber security / sci-fi interface style**.

It includes animated elements such as:

* AI telemetry panel
* Terminal log simulation
* Particle network background
* Glitch text effects
* Biometric authentication animation
* Cyber HUD design

The project is **front-end only** and intended for UI demonstration or future integration with a backend system.

## How to Run

1. Clone the repository

```
git clone https://github.com/your-username/kapaka-ai-console.git
```

2. Open the project folder

3. Run the file:

```
index.html
```

4. The interface will open directly in your browser.

No server setup is required.
